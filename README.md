# 🍽️ Landing Page - Restaurante

## 📌 Sobre o projeto

Este projeto consiste no desenvolvimento de uma **landing page moderna para restaurante**, com foco em apresentação visual, usabilidade e conversão de clientes.

A página foi construída utilizando **HTML, CSS e JavaScript puro**, com o objetivo de simular um site real de restaurante, destacando cardápio, ambiente e formas de contato.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 🎯 Objetivo

Criar uma interface atrativa e responsiva que:

* Apresente o restaurante de forma profissional
* Destaque os principais pratos
* Facilite o contato com o cliente
* Proporcione uma boa experiência em qualquer dispositivo

---

## 🖥️ Funcionalidades

* 📱 Layout totalmente responsivo
* 🍔 Seção de cardápio com destaque para pratos
* 🏠 Banner inicial chamativo (hero section)
* 📍 Informações de contato e localização
* 📩 Botão de ação (CTA) para reservas ou pedidos
* 🎨 Animações e interações com JavaScript
